Centre de Support & FAQ

Une question ? Un problème avec une commande ? Consultez notre foire aux questions ou contactez-nous directement.

C'est très simple ! Lors du paiement, choisissez "Solde Ooredoo". Entrez votre numéro de téléphone Ooredoo. Vous recevrez instantanément un code de vérification SMS à 4 chiffres sur votre téléphone (dans notre simulateur, un code vous sera affiché à l'écran). Saisissez ce code pour valider le débit direct sur votre solde de recharge.

La livraison est instantanée. Dès que la transaction est confirmée, votre code s'affiche à l'écran et vous est envoyé par SMS. Pour le rechargement direct (comme les diamants Free Fire par Player ID), le crédit est directement injecté sur votre compte de jeu en moins de 2 minutes.

L'application D17 est l'application officielle de la Poste Tunisienne. Si vous choisissez cette option, notre site affichera les informations de transfert (numéro de téléphone ou QR code de la boutique). Une fois le virement effectué, notre système valide le paiement pour délivrer votre commande.

Malheureusement, les rechargements directs par Player ID (comme Free Fire ou PUBG) sont irréversibles une fois validés sur le serveur de jeu. Veuillez vérifier attentivement votre identifiant de joueur avant de soumettre la commande.
